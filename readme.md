# vimrc

## About
**vimrc**,是自用的vim配置。使用[NeoBundle](https://github.com/shougo/neobundle.vim)作为插件管理器。


## 设置
**设置**，保存在mysetting目录中，主要是一些快捷键的设置和环境设置。

## 安装的插件：
